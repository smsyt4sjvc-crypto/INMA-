# raw/archive/ — the existing collection

Bulk drop zone for the user's pre-existing files: things they've **created** (memos, notes,
theses, journals) and things they've **saved** (articles, reports, PDFs). Upload the whole
collection here, then I inventory + triage it.

## Triage (how the collection gets processed)
- **CREATED by user** → the growth-review pile. I trace how the thinking evolved, score old
  calls against what happened, flag confirmation bias / thesis-dressed-as-fact, note blind
  spots. Output → `wiki/thinking-review.md`.
- **SAVED sources** → evidence pile. Ingested into the relevant `wiki/` notes behind the
  DATA/THESIS firewall, same as any source.

## Notes
- Text/markdown notes = cheap to read. Prioritized.
- Image/scanned PDFs = expensive in-container → may get digested via the
  [[data-sourcing-playbook]] offload instead of read whole.
- I list filenames first (near-zero cost) before reading anything, so we prioritize together.
